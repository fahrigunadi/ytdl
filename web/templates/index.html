GET /api/info

Returns available formats for a YouTube URL.

Query Parameters
ParamRequiredDescription
url yes YouTube video URL
Example Request

GET /download

Streams the video/audio file directly to the client. Use the download_url from /api/info — no need to construct this manually.

Query Parameters
ParamRequiredDescription
url yes YouTube URL, base64url-encoded
format_id yes Format ID from /api/info
ext yes File extension: mp4, webm, mkv, m4a, opus
title no Used as the downloaded filename
needs_merge no Set to 1 when needs_audio_merge is true (1080p+)
